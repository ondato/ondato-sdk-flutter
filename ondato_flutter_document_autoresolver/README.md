# ondato_flutter_document_autoresolver

The Ondato SDK Flutter plugin for automatic document capture. Works from v. 3.3.0 upwards. Previous versions will have this feature embedded inside the ondato_flutter package.

### Add dependency

Add `ondato_flutter_document_autoresolver` as a dependency in your pubspec.yaml file:

```
ondato_flutter:
    git:
      url: git@github.com:ondato/ondato-sdk-flutter.git
      ref: main
      path: ondato_flutter_document_autoresolver
```

For more details on how to use the SDK, please have a look at the Ondato Flutter Readme file.
